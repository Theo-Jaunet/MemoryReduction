Full Memory

We begin with a fully trained agent using 100% of its memory. During this interval, the agent moved towards the armor, and then towards where it started insted of reaching the health pack. This suggests that the agent forgot that it saw the health pack at the beginning.

Once the agent has gathered the armor in step 15 the first 3 elements change their activations from either active to inactive or the other way around.

The last element is inactive during this interval. How the agent would behave without it? Let's find out! We can see that the new trajectory is exactly the same as the previous one, thus we can conclude that is element has no impact in the agent's decision process during this sequence.

What if we go further and remove more memory elements? This would be useful to have smaller models which may be more interpretable as they may contain less memory dimensions, but also requiring less computing power and reduce the energy consumption footprint.
